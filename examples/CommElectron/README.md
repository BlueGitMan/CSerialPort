# CSerialPort for Electron based nodejs bindings

```
nodejs: 20.15.0 (2024-06-20)
cmake: 3.8.2
cmake-js: 7.3.0 (2024-01-16)
node-addon-api: 8.0.0 (2024-05-05)
electron: v31.0.2
```

## Install

```
npm install
```

## Build

```
cd examples/CommElectron
npm i -g cmake-js
npm run build
```

## Run

```
npm run start
```
